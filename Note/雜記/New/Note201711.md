### Linux Command
* 2017/11/12

查看環境變數
```bash
echo $PATH
```

添加到環境變數
```bash
echo "PATH=$PATH:/path/u/want/to/add" >> ~/.bashrc
```

這種方式只是暫時在當下的terminal裡添加到環境變數
```bash
export PATH=$PATH:/path/u/want/to/add
```
