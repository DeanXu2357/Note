### Linux Command
* 2017/11/8

顯示目前系統執行中的程序
``` 
ps ax 
```

顯示目前系統執行中的程序，一併顯示程序的擁有者
```
ps aux
```

查詢named現在是否執行中
```
ps ax | grep named
```
