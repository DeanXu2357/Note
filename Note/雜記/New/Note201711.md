### Linux Command
* 2017/11/8

顯示目前系統執行中的程序
``` 
ps ax 
```

顯示目前系統執行中的程序，一併顯示程序的擁有者
```
ps aux
```

查詢named現在是否執行中
```
ps ax | grep named
```

### Linux Command
* 2017/11/12

查看環境變數
```bash
echo $PATH
```

添加到環境變數
```bash
echo "PATH=$PATH:/path/u/want/to/add" >> ~/.bashrc
```

這種方式只是暫時在當下的terminal裡添加到環境變數
```bash
export PATH=$PATH:/path/u/want/to/add
```
